Hello, I’m Kareem Ahmed — a computer science student passionate about programming, problem solving, and creating meaningful digital experiences. My interests span web development, game development, and competitive programming.

I believe in learning through building. I started with small front-end projects, then moved on to frameworks like React.

I aim to create software that blends creativity, usability, and technical precision — whether it’s a web interface, a game, or an algorithmic solution.
